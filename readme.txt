WordPress plugin to charge your visitors for viewing posts and pages on your blog.

== Description ==

Free, fast to setup, easy to use solution to charge your blog readers for accessing
blog posts and pages. You will get the payments directly to your PayPal account and you can set the flat or recurring fee items.

== Installation ==

1. Upload `lockpress` directory from ZIP-archive (if you install from standalone ZIP file) to the `/wp-content/plugins/` directory (in the end it should look like /wp-content/plugins/lockpress/lockpress.php etc.)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Generate the code with lock generator

Latest and up-to-date manual is located at [LockPress site](http://twoenough.com/products/lockpress/manual)
